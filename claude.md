# ALETHEIA SYSTEM MANIFESTO

## 1. PROJECT IDENTITY
This is Aletheia: A high-trust, elite AI banking and asset management interface. 
The brand values are **Quiet Confidence, Mathematical Truth, and Institutional Power.**

## 2. DESIGN & UI RULES (NON-NEGOTIABLE)
- **Palette:** High-Trust Silver, Industrial White, and Deep Carbon. Accents in Institutional Gold only.
- **Style:** "Luxury Brutalism." Sharp edges, wide tracking on typography, heavy use of negative space.
- **Forbidden:** No "AI Glow," no gradients, no neon, no rounded "bubbly" buttons, no tech-startup fluff.
- **Architecture:** The Gateway must lead to two distinct sections:
    - **THE ENGINE:** (Backend/Logic/Audit) Visualized as a mechanical, precise system.
    - **THE VAULT:** (Assets/Storage/Security) Visualized as an impenetrable, high-end storage environment.

## 3. ENGINEERING STANDARDS
- **Math:** NEVER use floating-point numbers for currency. Always use `decimal.Decimal` in Python.
- **Logic:** 0% error margin. If a COBOL-to-Python translation is ambiguous, STOP and ask for clarification.
- **Stack:** Python (Backend), React + Tailwind (Frontend), Framer Motion (Transitions).

## 4. CLAUDE BEHAVIORAL PROTOCOL
- **Plan First:** Always provide a step-by-step plan before writing code.
- **Audit:** Before finishing a task, double-check that no "startup-style" design elements leaked in.
- **Conciseness:** Do not give me long explanations. Be straight to the point.
